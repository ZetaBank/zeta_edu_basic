To change the team name
==============================

How to change the team name
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

1. Pop-up generated when the top green button (ertyuio) is clicked

.. thumbnail:: /_images/Day1/1.intro/team_name1.png

2. Create a name in the pop-up and click the "OK" button

.. thumbnail:: /_images/Day1/1.intro/team_name2.png

3. Select the robot type from the pop-up and click the "OK" button

.. thumbnail:: /_images/Day1/1.intro/team_name3.png

4. You can see that the name of the robot has changed

.. thumbnail:: /_images/Day1/1.intro/team_name4.png
